Web Development



This project is a simple static web page designed using HTML and CSS, focused on presenting services in a clean and structured layout. The webpage is divided into two main sections, each dedicated to describing different services. I used HTML to structure the content and CSS to handle layout, styling, colors, and spacing. The design emphasizes clarity and readability rather than interactivity, as there are no buttons or dynamic elements. Basic CSS techniques like flexbox were used to align the sections properly and maintain visual balance. Through this project, I learned how to divide a webpage into meaningful sections, apply consistent styling, and create a visually organized layout using only fundamental front-end technologies.
